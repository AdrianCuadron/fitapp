# fitapp
Pagina web local dedicada al mundo fitness, que registra los productos consumidos y sus valores nutricionales.
Sirve para contar calorias que has consumido a lo largo del dia, incluyendo el registro de todo los productos consumisdos en el dia. Tambien da la opcion de registrar el peso en la fecha actual. Gracias a estos datos, cuando el usuario quiera, podra visualizar los datos registrados ordenados por fecha.
